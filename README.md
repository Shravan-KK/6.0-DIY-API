# DIY API

Small Node.js project for learning/experimenting with a DIY API.

## Contents
- index.js — application entry point
- solution.js — supplementary code or solution file
- package.json — project metadata and scripts

## Prerequisites
- Node.js (v14+ recommended)

## Install

```bash
npm install
```

## Run

Start with Node:

```bash
node index.js
```

Or use nodemon for live reload during development:

```bash
npx nodemon index.js
```
## Technologies used

- Express
- Body parser Middleware
